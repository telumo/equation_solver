EQUEATION SOLVER
--

## What's this?

This is a web app that solves multivariate linear equations.

## How To Use

## For developer

### 🛠️ Build

When building for the first time, ensure to install dependencies first.

```sh
$ yarn install
```

```sh
$ yarn run build
```

### 🔬 Serve locally

```sh
# start app & open browser
$ yarn run start:dev
```
